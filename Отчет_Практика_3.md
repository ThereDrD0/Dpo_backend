# Отчет по практической работе 3

Проект: StudentPortal.Diagnostics на ASP.NET Core (.NET 10).

Проверенные запросы:

- `/` - стартовая страница со списком маршрутов.
- `/tools/time` - текущее время через `IDateTimeService`.
- `/tools/date` - текущая дата через `IDateTimeService`.
- `/tools/info` - краткая информация о ветке `/tools`.
- `/tools/time?trace=true` - сначала проходит ветку `UseWhen`, добавляется заголовок `X-Debug-Trace`.
- `/anything?format=plain` - запрос перехватывает `MapWhen`, основной маршрут дальше не выполняется.
- `/secure/report` - без токена возвращается `403 Forbidden`.
- `/secure/report?token=study2026` - с правильным токеном доступ разрешен.
- `/env` - выводит `EnvironmentName`, `ApplicationName`, `ContentRootPath`, `WebRootPath`.
- `/di/services` - показывает количество сервисов и список зарегистрированных сервисов.
- `/unknown` - возвращает `404 Not Found`.

Порядок middleware:

Сначала стоит `ErrorHandlingMiddleware`, чтобы после выполнения следующих компонентов увидеть итоговый `StatusCode` и дописать понятный текст для `403` и `404`. После него идет inline middleware через `Use`, который пишет начало и конец обработки запроса и добавляет заголовок `X-Request-Time-Ms`.

`UseWhen` используется для `trace=true`. Эта ветка добавляет диагностику, но потом возвращает запрос в основной конвейер обработки запроса (pipeline). `MapWhen` используется для `format=plain`: это отдельная ветка, она сама пишет ответ и не возвращается к обычным маршрутам.

Собственные сервисы:

- `IDateTimeService` и `DateTimeService` - дата и время.
- `IEnvironmentReportService` и `EnvironmentReportService` - информация об окружении.

Они регистрируются в `AddStudentPortalServices()` до `builder.Build()`.

Среда запуска задается в `Properties/launchSettings.json` через `ASPNETCORE_ENVIRONMENT=Development`. В `/env` видно, что приложение запущено в `Development`.

Главный вывод: порядок middleware важен, потому что каждый компонент либо передает запрос дальше через `next`, либо останавливает обработку и сам формирует ответ.
